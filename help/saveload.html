Saving and Loading Simulations

Evolve allows for multiple documents to be loaded at the same time.

To Save:

Press the tool bar button to save the current simulation. Use "Save As..." from the menu to save the simulation with a different name. duh!

To Load:

Press the tool bar button to load a new simulation.

To Create a New Simulation:

Press the tool bar button to create a blank simulation.

To create a new simulation using the New Universe dialog, go to the menu File >> New.

ASCII Formatted Files:

The simulation files used by Evolve 5.0 are ascii text files. The .txt extension is often used for Evolve simulation files.

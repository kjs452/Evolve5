Evolve 5.0 Source Code

The Evolve source code is available on github: https://github.com/kjs452. The repository names are:

  • Evolve5: The MacOS Xcode project for the Evolve 5.0 GUI.

  • Evolve5Batch: The command line utility to run the simulator without any GUI. multi-platform.

  • Evolve5Help: The help files, unit tests, scripts, instruction documentation, static site generation, Tagged{Text}.
